## 🍽프로젝트 소개  
점심을 시켜먹고 싶은데  
- 최소 주문 금액이 높아 1인분만 시킬 수가 없어서,    
- 배달비가 아깝거나 가격이 비싸서, 때론 양이 너무 많아서  
- 메뉴를 바꾸거나 망설인적이 있으신가요?  
  
그런 분들을 위한 배달 모임 서비스 밀착🍙입니다!    
밀착으로 같이 배달을 시킬 친구👭를 찾아 보세요.

- [노션으로 가기](https://bit.ly/mealchak-notion)
